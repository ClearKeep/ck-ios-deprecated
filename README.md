# ck-ios
iOS client with end to end encryption messaging

# Prerequisites
* Xcode 12.0.1
* Swift-protobuf version 1.12.0
* grpc-swift commit: e2e138df61dcbfc2dc1cf284fdab6f983539ab48

## Build & Run

* Git clone source code
* Pod update 
* Using Xcode import this project
* Tab run app (cmd + R)
